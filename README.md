# hello-world
Porting hello world to every platform

## Seting up repo
```
git clone https://github.com/tumble1999/hello-world.git`
cd hello-world
chmod +x ./scripts/gitflow.sh
./scripts/gitflow.sh
```
and then to start development on something
```
git flow feature start platform-name
```
